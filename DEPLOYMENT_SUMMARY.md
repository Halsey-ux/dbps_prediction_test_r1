## 🎯 使用指南

### 快速开始
```bash
# 1. 克隆项目
git clone https://github.com/Halsey-ux/dbps_prediction_test_r1.git
cd dbps_prediction_test_r1

# 2. 设置环境（方法A：Conda）
conda env create -f environment.yml
conda activate test_r1_env

# 3. 启动应用
python run_app.py
```

### 本地访问
🔗 **应用地址**: http://localhost:8501
